# Freshippo Skill v2.0

盒马鲜生购物助手 Skill，支持浏览器自动化完成搜索、生鲜选购、加购物车等操作，支付环节留给用户控制。

## 功能特性

### v2.0 - 新增浏览器自动化

- 🔍 **商品搜索**：关键词搜索，支持生鲜筛选
- 🥬 **生鲜选购**：读取产地、溯源码、新鲜度指标
- 💰 **价格对比**：普通价 vs X会员价对比
- 🛒 **购物车操作**：登录态下添加商品
- 🎫 **X会员权益**：自动检查会员折扣
- 🚚 **配送时段**：查看30分钟配送 slot
- 📋 **订单预览**：生成完整订单信息（不含支付）

### v1.0 - 决策支持

- ✅ 生鲜选购指南
- ✅ 配送时段优化
- ✅ X会员权益详解
- ✅ 一周买菜规划

## 安全边界

| 操作 | Agent | 用户 |
|------|-------|------|
| 搜索/浏览 | ✅ | - |
| 查看新鲜度 | ✅ | - |
| 加购/用券 | ✅ | - |
| **支付/提交订单** | ❌ | ✅ |

## 工作流程

```
发现 → 对比 → 选择 → 加购 → 选配送 slot → 预览 → [用户支付]
```

## 版本历史

- v2.0.0 - 新增浏览器自动化支持
- v1.0.0 - 基础购物指南

## 安装

```bash
clawhub install freshippo
```

```
用户：帮我买盒马的三文鱼
Agent：搜索 → 对比新鲜度 → 加购 → 选配送 slot → 生成订单预览
用户：👤 完成支付
```
